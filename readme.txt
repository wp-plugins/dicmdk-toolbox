== Description ==

This toolbox contains a large number of tools and scripts for website owners. More tools are added.

Current and Future features:
* License key generator (If you need your own license keys for something)
* Antispam (Several types)
* Antihacking - Several types
* Return To Login Fix
* Login block - By country, ip or other. Bypass if you get locked out and is a payed license user.
* Two step login security (payed users only)
* QR code generator
* CSS optimizer (same as the one we use on your site, this is just from within the plugin)

This plugin is free but some features requires payed license.
The free license can be optained at our website by using the contact form.

* Note: This plugin uses our API function for a lot of the scripts, all data transfered between the API and your site is currently not in HTTPS.

== Installation ==

1. Upload `dicmdk-toolbox` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Optain a license at dicm.dk (contact form)
